## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/aryankuttarmare14/iamaryan.dev.git
cd iamaryan.dev
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the dev server

```bash
npm run dev
```

Now open http://localhost:5173 in your browser.
